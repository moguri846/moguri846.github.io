# 자석의 자기작용   

```
   전기     vs      자기   
기전력 E[V]    기자력   F = NI[AT]   
전류   I[A]    자속     Φ [wb]
저항   R[Ω]    자기저항 Rm [AT/wb]
유절율 ε[F/m]  투자율   μ[H/m]   
ε = ε0 εs      μ = μ0 μs

ε0 = 8.855 * 10^-12
F = 9 * 10^9 * Q1 Q2/r^2

μ0 = 4𝝅 *  10^-7   
F = 6.33 * 10^4 * m1 m2/r^2
```
