# 슬립   

S = Ns - N/Ns * 100%   

S = 1, N = 0 **정지**   

S = 0, Ns = N **동기속도로 회전** 가장 이상적   

**0 < S < 1**     

N = (1 - S)Ns   
 = (1 - S) 120*f/P   
 = ηNs   

η = N/Ns
