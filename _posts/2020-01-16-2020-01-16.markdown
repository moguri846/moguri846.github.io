# 속도와 토크   

## 발전기   

유기기전력   

V = E - IaRa   

## 전동기   

역기전력   

Ec = V - IaRa   

N = V-IaRa/Φ [V]   

속도를 증가   

   + V : 전압 증가  (유효)
   + Φ : 자속 감소 -> 정출력
   + Ra: 저항 감소   

계자저항 증가 -> If 감소 -> Φ 감소 -> N 증가   

# 제동   

   + 발전제동   
   + 회생제동   ★
   + 역상제동   

# 토크  

P = wて  
て = P/W = P/2 π/n60   
 = 0.975 P/N [Kg * m]   

분권   

T ∝ 1/N   

직권   

T ∝ 1/N^2
