# 단권 변압기   

자기 용량 = (V2 - V1)I2

부하 용량 = V2I2   

자기 용량/부하 용량    
 = (V2 - V1)I2/V2I2   
 = V2 - V1/V2   
 = Vh - Vl = Vh
